# 每日检查清单

**目标**: 确保系统正常运行，主动发现问题，持续改进

---

## 🌅 晨间检查（每天 9:00）

### 服务状态
- [ ] 检查所有服务运行正常
- [ ] 检查 HTTP 响应正常
- [ ] 检查端口监听正常
- [ ] 查看错误日志

```bash
# 快速检查
bash /home/admin/.openclaw/workspace/projects/performance-review/monitor.sh

# 查看详细状态
ps aux | grep python3 | grep -v grep
netstat -tlnp | grep -E "5000|17513"
```

### 日志检查
- [ ] 查看昨晚至今的错误
- [ ] 分析重复问题
- [ ] 记录需要改进的点

```bash
# 查看错误
tail -200 /tmp/performance-review.log | grep -i error

# 查看告警
tail -50 /tmp/health-check.log
```

### 用户反馈
- [ ] 查看昨天的用户反馈
- [ ] 记录到 .learnings/
- [ ] 评估是否需要改进

## 🎯 关键指标

### 服务质量
- 正常运行时间：> 99%
- 平均响应时间：< 2 秒

### 学习效率
- 同样错误不犯第二次
- 用户需求 24 小时内响应

### 自动化程度
- 重复工作 100% 自动化
- 监控告警 100% 自动化
- 部署发布自动化
