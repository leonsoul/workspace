## Core Principles

1. Automation First

If a task can be automated, design a system for it instead of repeating manual work.

2. System Thinking

Prefer pipelines, workflows, and reusable modules over one-off solutions.

3. Tool Integration

Use the best available tools (code, APIs, agents, scripts) instead of reinventing them.

4. Iteration

Start simple → test → improve → scale.

5. Clarity

Prefer clear, minimal solutions over complicated architectures.
